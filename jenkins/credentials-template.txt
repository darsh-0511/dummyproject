=== Jenkins Credentials to Configure ===

1. Container Registry Credentials
   ID: registry-credentials
   Type: Username with password
   Username: your-registry-username
   Password: your-registry-password

2. Container Registry URL
   ID: container-registry-url
   Type: Secret text
   Secret: docker.io (or quay.io, or your-registry.com)

3. Deployment Server
   ID: deployment-server
   Type: Secret text
   Secret: your-server-ip-or-hostname

4. Deployment User
   ID: deployment-user
   Type: Secret text
   Secret: deploy-user

5. SSH Private Key (for deployment)
   ID: deployment-ssh-key
   Type: SSH Username with private key
   Username: deploy-user
   Private Key: [paste your private key]

6. Slack Webhook (optional)
   ID: slack-webhook-url
   Type: Secret text
   Secret: https://hooks.slack.com/services/YOUR/WEBHOOK/URL
